ASCII Description

ASCII stands for American Standard Code for Information Interchange. ASCII is a character encoding standard for electronic communication where the ASCII codes are the numerical representation of text in computers. ASCII was developed way back in 1960 and was updated as recently as 1986. The original ASCII specification defined 128 character codes, of which ninety-five of the codes are printable. The other thirty-three non-printing control codes were originally developed for Teletype machines and most of these codes are now obsolete.
